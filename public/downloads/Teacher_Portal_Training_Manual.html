Lesson 1.1

Logging In & First-Time Setup

Objective: Successfully log into the portal and change your default password.

Accessing the Portal

  1. Open your web browser (Chrome, Firefox, or Edge recommended)
  2. Type the portal address: https://portal.stfrancisofassisizm.com/admin
  3. Enter your Email Address provided by the school administrator
  4. Enter your Password (default password is given by admin)
  5. Click "Sign In"

First-Time Password Change

When you log in for the first time, the system will force you to change your password. This is a security measure to ensure only you can access your account.

  • Your new password must be at least 8 characters long
  • Use a mix of letters, numbers, and special characters for a strong password
  • Do not reuse passwords from other websites
  • Remember your password — write it down in a secure place if needed
Security: Never share your login credentials with anyone. Each teacher has their own unique account. If you suspect someone has your password, change it immediately.

Lesson 1.2

Understanding Your Role

Objective: Understand the difference between Class Teacher and Subject Teacher roles and their permissions.

Two Types of Teacher Roles

RoleDescriptionKey Permissions
Class Teacher Assigned as the main teacher responsible for a specific class section (e.g., Grade 3A) • Mark attendance for the class
• Enter results for ALL subjects
• Write class teacher comments on report cards
• View all students in the class
Subject Teacher Teaches specific subjects in one or more classes (e.g., Mathematics in Grade 7A & 7B) • Enter results for assigned subjects ONLY
• Create homework for assigned subjects
• View students in classes they teach
• Cannot write class teacher comments
Note: A teacher can be BOTH a class teacher for one class AND a subject teacher in other classes. For example, you may be the Class Teacher for Grade 5A while also teaching Science in Grade 5B and 6A.

What Teachers Cannot Do

  • Access financial or accounting modules (fees, payments)
  • Manage the school timetable (admin function)
  • Edit student personal details (admin function)
  • Access other teachers' records or payslips
  • Change school settings or configurations
  • Delete other teachers' homework or results

Lesson 2.1

Marking Daily Attendance

Objective: Mark attendance for your class accurately and efficiently.

Step-by-Step: How to Mark Attendance

  1. Select Class — If you are a class teacher, your class is auto-selected. Subject teachers select from a dropdown of their assigned classes.
  2. Select Date — Defaults to today. You can select a past date for missed days. Future dates are blocked.
  3. Click "Mark All Present" — This marks all students as Present in one click (saves time).
  4. Adjust Individual Students — Click the status button next to students who are absent, sick, late, or excused. Each click cycles to the next status.
  5. Click "Submit Attendance" — A green notification shows the summary: Present: X | Absent: X | Sick: X | Late: X | Excused: X

Attendance Status Options

StatusSymbolWhen to UseColour on Button
PresentPStudent is in schoolGreen
AbsentXStudent is not in school, no reason givenRed
SickSStudent is sick (medical or reported by parent)Blue
LateYStudent arrived after official start timeYellow
ExcusedLPre-approved absence (family event, appointment)Grey
Time-Saving Tip: Always start by clicking "Mark All Present" first, then adjust only the few students who are absent, sick, or late. This is much faster than marking each student individually.

Important Rules

  • Attendance should be marked every school day by the class teacher
  • Mark attendance before the first period (ideally before 08:00)
  • You can edit attendance for the same day by going back and resubmitting
  • You cannot mark future dates
  • The system records the exact time attendance was submitted

Lesson 3.1

Entering Student Results

Objective: Enter student marks correctly for any exam type.

Step-by-Step: Entering Results

  1. Go to Enter Results — Sidebar > Academic > Enter Results
  2. Select Class — Choose the class section from the dropdown
  3. Select Subject — Only shows subjects assigned to the selected grade (and your teaching assignment)
  4. Select Term, Exam Type & Year — Choose the correct combination
  5. Enter Marks — Type marks (0-100) for each student. Use Tab to move between fields.
  6. Review Grades — System auto-calculates grades. Override manually if needed.
  7. Add Comments — Optional feedback per student (e.g., "Well done", "Needs more practice")
  8. Submit — Click "Submit Results". System shows saved count.
Important: If a student was absent for the exam, leave their marks blank. Do NOT enter 0, as 0 means the student sat the exam and scored zero.

Lesson 3.3

Report Cards: Generating, Commenting & Distributing

Objective: Generate report cards, write meaningful comments, and distribute to parents.

Step-by-Step: Generating Report Cards

  1. Verify All Results — Ensure ALL subjects have marks entered. Check the Results Count column matches the Subject Count.
  2. Go to Generate Report Cards — Sidebar > Academic > Generate Report Cards
  3. Select Class & Term — Choose your class and the current term
  4. Add Class Teacher Comments — Click on each student to write personalised comments
  5. Generate — Click "Generate" for individual or "Generate All" for the entire class
  6. Review — Open a few report cards to check accuracy

Writing Effective Comments

Good CommentsPoor Comments
"An excellent term. Shows strong analytical skills in Mathematics. Should continue reading widely.""Good"
"Has improved significantly in English. Needs to work on time management for assignments.""OK"
"A dedicated student who participates actively. Must practise more in Science practicals.""Average"

Distributing via SMS

You can notify parents via SMS after report cards are ready. The SMS includes the student's name, rank, and top subject marks. This requires available SMS credits.

Lesson 5.3

Best Practices, Security & Troubleshooting

Objective: Follow security best practices and know how to resolve common issues.

Security Best Practices

#PracticeWhy It Matters
1Never share your passwordYour account tracks all your actions. Sharing gives others access to student data.
2Log out on shared computersPrevents the next person from accessing your account.
3Use a strong passwordWeak passwords are easily guessed. Use letters, numbers, and symbols.
4Change password regularlyReduces risk if your password is compromised.
5Don't save passwords in public browsersBrowser-saved passwords on shared computers are a security risk.

Efficiency Best Practices

#Practice
1Mark attendance daily before the first period
2Enter results promptly after each assessment, not at end of term
3Double-check marks before submitting
4Write meaningful, personalised comments on report cards
5Keep your profile information current
6Use Chrome or Firefox for the best experience

Common Issues & Solutions

ProblemSolution
Can't see my class in dropdownContact admin to assign you via Teacher Assignments
Can't enter results for a subjectYou need to be assigned as subject teacher. Contact admin.
Report card shows 0 resultsResults must be entered via Enter Results first. Check term/year.
Forgot my passwordContact the administrator for a password reset
Page not loading properlyTry refreshing (Ctrl+F5), clearing browser cache, or switching to Chrome
SMS failed to sendSMS credits may be exhausted. Contact admin to top up.
