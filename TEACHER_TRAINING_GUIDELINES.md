## STEP-BY-STEP PROCEDURES

### PROCEDURE 1: Recording Daily Attendance

**Time Required:** 2-5 minutes per class

1. **Navigate to Attendance**
   - Click **"Attendance"** in the main menu
   - Select your class from the dropdown
   - Select today's date

2. **Mark Attendance**
   - You will see a list of all students in the class
   - For each student, select their status:
     - ✅ **Present** - Student is in class
     - ❌ **Absent** - Student is not in class
     - ⏰ **Late** - Student arrived late
     - 📝 **Excused** - Student has valid excuse

3. **Quick Options**
   - Use **"Mark All Present"** button to mark everyone present
   - Then manually change status for absent/late students

4. **Save Attendance**
   - Click **"Submit Attendance"**
   - Confirm the submission
   - Parents will be notified automatically

5. **Best Time to Record**
   - Record attendance at the start of each class
   - For primary classes: Record once daily (morning)
   - For secondary classes: Record per subject period

---

### PROCEDURE 2: Creating Homework/Assignments

**Time Required:** 5-10 minutes

1. **Navigate to Homework**
   - Click **"Homework"** or **"Assignments"** in the menu
   - Click **"Create New Assignment"** button

2. **Enter Assignment Details**
   - **Title:** Give a clear, descriptive title
     - Example: "Mathematics - Chapter 5 Exercises"
   - **Subject:** Select the subject
   - **Description:** Provide detailed instructions
   - **Due Date:** Set the deadline
   - **Total Marks:** Enter maximum points (if graded)

3. **Select Recipients**
   - Choose the class(es) to receive the assignment
   - Or select specific students (optional)

4. **Attach Resources (Optional)**
   - Click **"Add Attachment"**
   - Upload files, worksheets, or images
   - Supported formats: PDF, DOC, DOCX, JPG, PNG

5. **Set Submission Type**
   - Online submission (students upload files)
   - Physical submission (mark as submitted manually)
   - No submission required (reading/practice only)

6. **Save and Publish**
   - Review all details
   - Click **"Save as Draft"** (to publish later)
   - Or click **"Publish"** (students see immediately)
   - Students and parents receive notifications

---

### PROCEDURE 3: Entering Student Grades

**Time Required:** 10-20 minutes per class

1. **Navigate to Results/Grades**
   - Click **"Results"** or **"Grades"** in the menu
   - Select your **Class**
   - Select **Subject**
   - Select **Assessment Type** (Test, Quiz, Exam, etc.)

2. **Choose Grade Entry Method**

   **Option A: Individual Entry**
   - Click on each student
   - Enter their score
   - Add comments (optional)
   - Click **"Save"**

   **Option B: Bulk Entry (Faster)**
   - You'll see a spreadsheet-like view
   - Enter all scores in the grid
   - Tab or Enter to move between cells
   - Click **"Save All"** when done

3. **Enter Score Details**
   - **Score:** Enter the marks obtained
   - **Total Marks:** Confirm maximum marks
   - **Percentage:** Calculated automatically
   - **Grade:** Assigned automatically (A, B, C, etc.)
   - **Comments:** Add feedback (optional but recommended)

4. **Review and Submit**
   - Check for any errors or missing entries
   - Use **"Preview"** to see how it will appear
   - Click **"Submit Grades"**
   - Confirm submission (parents will be notified)

5. **Grade Publishing**
   - Grades can be saved as draft (not visible to students)
   - Publish when ready for students/parents to view
   - You can unpublish and edit if needed

---

### PROCEDURE 4: Viewing Your Schedule

**Time Required:** 1-2 minutes

1. **Access Your Timetable**
   - Click **"Schedule"** or **"My Timetable"**
   - Default view shows current day

2. **View Options**
   - **Daily View:** See today's classes
   - **Weekly View:** See entire week at a glance
   - **Monthly View:** See the full month

3. **Schedule Information Displayed**
   - Time slot
   - Subject/Class
   - Room number
   - Grade/Level
   - Number of students

4. **Print Your Schedule**

   - Select weekly or monthly format
   - Print or save as PDF

---

### PROCEDURE 5: Communicating with Parents

**Time Required:** 3-5 minutes per message

1. **Navigate to Communications**
   - Click **"Messages"** or **"Communication Center"**
   - Click **"New Message"**

2. **Select Recipients**

   **Option A: Individual Parent**
   - Click **"Select Parent"**
   - Search by parent name or student name
   - Select the parent

   **Option B: Class Parents**
   - Select **"Send to Class"**
   - Choose your class
   - All parents in that class selected

   **Option C: Multiple Parents**
   - Use checkboxes to select multiple parents
   - Or use **"Select All"**

3. **Compose Message**
   - **Subject:** Clear subject line
     - Example: "Grade 3A - Parent-Teacher Meeting Reminder"
   - **Message:** Write your message
     - Be clear and professional
     - Include all necessary details
     - Add action items if needed
   - **Attachments:** Add files if needed (optional)

4. **Send Message**
   - Click **"Send"**
   - Parents receive notification via email/SMS
   - Message appears in their parent portal

5. **View Sent Messages**
   - Click **"Sent Messages"** to see history
   - Track who has read your messages
   - Reply to parent responses

### PROCEDURE 6: Generating Reports

**Time Required:** 2-5 minutes

1. **Navigate to Reports**
   - Click **"Reports"** in the main menu
   - Browse available report types

2. **Available Report Types**
   - **Attendance Reports:** Class or individual student attendance
   - **Grade Reports:** Student performance by subject
   - **Class Performance:** Overall class statistics
   - **Assignment Reports:** Homework completion rates
   - **Custom Reports:** Build your own report

3. **Generate a Report**
   - Select **Report Type**
   - Set **Parameters:**
     - Class
     - Date Range
     - Subject (if applicable)
     - Student (for individual reports)
   - Click **"Generate Report"**

4. **View and Export**
   - Report displays on screen
   - **Export Options:**
     - PDF (for printing)
     - Excel (for further analysis)
     - CSV (for data import)
   - Click **"Download"** or **"Print"**

## BEST PRACTICES

### Attendance
✅ **DO:**
- Record attendance daily at the same time
- Mark attendance within first 10 minutes of class
- Double-check before submitting
- Contact absent students' parents

❌ **DON'T:**
- Forget to record attendance (parents rely on it)
- Mark students present who are absent
- Leave attendance unmarked for days

### Homework/Assignments
✅ **DO:**
- Give clear, specific instructions
- Set reasonable due dates (minimum 2-3 days)
- Provide resources and examples
- Give feedback on submitted work
- Use varied assignment types

❌ **DON'T:**
- Create unclear or vague assignments
- Set unrealistic deadlines
- Ignore submitted work without feedback
- Overload students with too many assignments

### Grading
✅ **DO:**
- Enter grades within 1 week of assessment
- Add meaningful comments
- Double-check scores before publishing
- Use the grading rubric consistently
- Inform students of grading criteria beforehand

❌ **DON'T:**
- Delay grade entry (parents expect timely updates)
- Enter incorrect scores
- Publish without reviewing
- Change grades without documentation

### Communication
✅ **DO:**
- Respond to parent messages within 24 hours
- Be professional and courteous
- Provide specific examples when discussing student behavior
- Keep records of all communications
- Use positive language when possible

❌ **DON'T:**
- Ignore parent messages
- Use informal or unprofessional language
- Share confidential information with unauthorized persons
- Send messages late at night or on weekends (unless urgent)

### Security
✅ **DO:**
- Keep your password confidential
- Log out when using shared computers
- Change your password regularly
- Report suspicious activity immediately

❌ **DON'T:**
- Share your login credentials
- Leave your account logged in on public computers
- Use simple, easy-to-guess passwords
- Let students access your teacher account

## TROUBLESHOOTING

### Common Issues and Solutions

#### Problem 1: Cannot Login
**Symptoms:** "Invalid email or password" error

**Solutions:**
1. ✅ Verify you're using the correct email address
2. ✅ Check if Caps Lock is ON (passwords are case-sensitive)
3. ✅ Try resetting your password:
   - Click "Forgot Password"
   - Enter your email
   - Check email for reset link
4. ✅ Clear browser cache and cookies
5. ✅ Try a different browser
6. ✅ Contact IT support if still unable to login

#### Problem 2: Cannot See My Classes
**Symptoms:** No classes appear in dropdown menus

**Solutions:**
1. ✅ Verify you've been assigned to classes (contact admin)
2. ✅ Refresh the page (F5 or Ctrl+R)
3. ✅ Log out and log back in
4. ✅ Check if you're in the correct academic year/term
5. ✅ Contact IT support

#### Problem 3: Cannot Upload Files
**Symptoms:** Error when attaching files to assignments

**Solutions:**
1. ✅ Check file size (maximum 5MB per file)
2. ✅ Verify file format is supported (PDF, DOC, DOCX, JPG, PNG)
3. ✅ Try a smaller file or compress it
4. ✅ Check your internet connection
5. ✅ Try uploading one file at a time
6. ✅ Use a different browser

#### Problem 4: Grades Not Saving
**Symptoms:** Entered grades disappear after saving

**Solutions:**
1. ✅ Check if you clicked "Save" or "Submit"
2. ✅ Ensure all required fields are filled
3. ✅ Check if grade is within valid range (0 to max marks)
4. ✅ Try entering grades one at a time
5. ✅ Take a screenshot before submitting
6. ✅ Report to IT support with details

#### Problem 5: Page Loads Slowly
**Symptoms:** Portal is very slow or unresponsive

**Solutions:**
1. ✅ Check your internet connection speed
2. ✅ Close unnecessary browser tabs
3. ✅ Clear browser cache and cookies
4. ✅ Restart your browser
5. ✅ Try during off-peak hours (early morning/late evening)
6. ✅ Contact IT if problem persists

#### Problem 6: Cannot See Students' Submissions
**Symptoms:** Assignment shows no submissions but students claim they submitted

**Solutions:**
1. ✅ Check the correct assignment is selected
2. ✅ Verify submission deadline hasn't passed (late submissions may need approval)
3. ✅ Refresh the page
4. ✅ Ask student to check their "Submitted Assignments" section
5. ✅ Contact IT support to verify submission
